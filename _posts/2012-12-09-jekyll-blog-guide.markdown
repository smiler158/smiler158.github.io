   mac下jekyll环境搭建只需要以下几条命令：

1.   gem  install jekyll

2.   jekyll new  mysite

3.   cd mysite

4.   jekyll   server

然后浏览器打开 http://localhost:4000  就可以看到博客页面了。
其他帮助查看<https://github.com/jekyll/jekyll>，代码可以托管到github上面。
